# cx-data-mock

### 介绍
根据SQL生成数据。
使用springboot2.1.1.RELEASE集成JSqlParser 
利用JSqlParser 解析SQL语句，生成数据。
官网地址: https://jsqlparser.sourceforge.net/
官网API: https://jsqlparser.sourceforge.net/docs/
